
# Dashboard de Previsão de Evasão de Clientes

O que este Dashboard oferece:

## Filtros Dinâmicos: Você pode filtrar os dados por País, Gênero e Tipo de Cartão na barra lateral, e todos os gráficos se atualizam automaticamente.

## KPIs em Tempo Real: Resumo do total de clientes, taxa de churn e satisfação média.

## Análise Geográfica: Visualização de quais países perdem mais clientes (Alemanha costuma ter taxas maiores neste dataset).

## Perfil Demográfico: Histograma para entender em qual faixa etária o churn é mais crítico.

## Interatividade Plotly: Todos os gráficos permitem dar zoom, filtrar legendas e ver detalhes ao passar o mouse.
